This is an unofficial, showcase personal trainer website created for demonstration purposes.

The entire website was built using **HTML, SCSS/CSS, and JavaScript**.

- Modern, responsive design
- Section and element entrance animations
- Parallax effect in the hero section
- Animated statistics
- Interactive offer highlights
- Contact form
- Client testimonials section
